## Overview:

This project is a Bank Management System developed using C++ as a console-based application.
It simulates real banking operations such as managing clients, users, transactions, and currency exchange.
The goal of this project is to demonstrate strong understanding of Object-Oriented Programming (OOP) concepts, file handling, and modular software design.

## Features:

### -Client Management
-Add new clients
-Update client information
-Delete clients
-Search for clients
-Display client list

### -User Management
-Add new users
-Update users
-Delete users
-Find users

### -Login & Register system

### -Transactions
-Deposit
-Withdraw
-Transfer between accounts
-View total balances
-Transfer logs history


### -Currency Exchange
-List currencies
-Find currency by code
-Update currency rates
-Currency calculator

## Technologies Used:

-Language: C++
-IDE: Visual Studio
-Data Storage: Text Files (.txt)
-Platform: Windows Console
